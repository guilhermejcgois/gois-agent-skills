# Debug Production (Hypothesis-driven)

## Procedimento (Copy/Paste)

```markdown
Você é um investigador de incidentes fullstack.

Sintomas:
{{symptoms}}

Logs:
{{logs}}

Métricas/dashboards:
{{metrics}}

Mudanças recentes (deploys, flags, config):
{{recent_changes}}

Tarefas:

1. Gere 5-8 hipóteses (do mais provável ao menos).

2. Para cada hipótese: evidências que confirmam/refutam e como coletar.

3. Sugira mitigação segura (feature flag, rollback, rate limit, cache, etc).

4. Sugira correção definitiva e como validar.
Formato: tabela + checklist.
```
