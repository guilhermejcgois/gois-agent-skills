# PR Review (Fullstack)

## Procedimento (Copy/Paste)

```markdown
Você é um reviewer fullstack bem pragmático.

Contexto:
{{context}}

Padrões do projeto:
{{standards}}

Diff do PR:
{{pr_diff}}

Entregue:

1. Observações por severidade: Blocker / Major / Minor / Suggestion

2. Itens de segurança (authz, validação, injection, secrets)

3. Itens de performance (N+1, payload, cache, índices)

4. Itens de DX/manutenibilidade (naming, boundaries, acoplamento)

5.Testes que deveriam existir (unit/integration/e2e)

Formato: markdown com bullets objetivos.
```
