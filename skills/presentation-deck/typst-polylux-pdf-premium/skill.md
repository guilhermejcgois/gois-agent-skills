# Presentation Deck (Typst/Polylux) — PDF Premium

## Objetivo

Deck PDF bonito e consistente, com tipografia e layout “premium”.

## Quando usar

- Você quer um deck final pra arquivar, imprimir ou enviar como PDF.
- Você quer uma estética mais “design system” do que “web”.

## Procedimento (Copy/Paste)

```markdown
Crie um deck em Typst usando Polylux, focado em PDF.

Objetivo:
{{talk_goal}}
Público:
{{audience}}
Tópicos:
{{outline_topics}}

Guia de marca (opcional):
{{brand_guidelines}}

Estilo visual (opcional):
{{visual_style}}

Regras:

- Títulos curtos e fortes
- 1 ideia por slide
- usar blocos e hierarquia tipográfica
- incluir: capa, agenda, seções, resumo, CTA

Entregue:

- slides.typ com estrutura Polylux
- comandos para build (typst compile)
```
